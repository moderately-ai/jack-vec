# Versioned results

Authoritative platform summaries generated by `../tools/run_matrix.py` are kept
here as stable JSON and Markdown filenames. Git history provides the timeline.

The `raw/` child directory is intentionally ignored because it contains every
per-round Criterion estimate and allocation CSV. Preserve those directories as
CI or experiment artifacts when investigating a result.

Exploratory runs are automatically suffixed `-exploratory` and must not replace
an authoritative platform summary.
